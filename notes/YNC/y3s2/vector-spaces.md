# Vector Spaces

There are two important operations within a vector space which contains a set $V$ and a field $\mathbb{F}$ which itself

1. Addition : Where we take as input **two elements of a set V** and return a single elemnt of a set V
2. Scalar Multiplication : Where we take as input **one scalar and one element of a set V** and return a single element of a set V.

We want our operations to be closed under addition and scalar multiplication. This means that a sum of two elements in a set must belong to the original set. The same should also hold for a scalar multiple of an eleemtn of the set. When this holds, we say that $V$ is closed under addition and closed under scalar multiplication.

## Vector Field Properties

1. Addition is commutative

   $$
   \forall u,v \in V , u + v = v + u
   $$

2. Addition is associative

$$
\forall u,v,w \in V, u + (v+w) = (u+v)+ w
$$

3. The set V contains an additive identity

$$
\exists \epsilon \in \mathbb{V} \text{ such that } \forall v \in V , v + \epsilon = v
$$

4. The set V contains an additive inverse

$$
\forall v \in V , \exists u \in V \text{ such that } u + v = \epsilon
$$

5. Scalar multiplcation is associative

$$
\forall \alpha,\beta \in \mathbb{F}, v \in V, \alpha\cdot(\beta v) = \alpha \cdot \beta (v)
$$

6. The field $\mathbb{R}$ contains a multiplicative identity such that $\forall v \in V, v \cdot 1 = v$

7. Operations interact via distributive property.

$$
\forall u,v \in V \& \alpha \in \mathbb{F}, \alpha ( u + v) = \alpha \cdot u + \alpha \cdot v
$$

$$
\forall \alpha, \beta \in \mathbb{F} \& v \in V, \alpha\cdot v + \beta \cdot v = (\alpha + \beta) \cdot v
$$

Note

1. Any field $\mathbb{F}$ is a vector space over itself. We can also define a set of vectors using the existing field by creating a set of tuples of length $n$ containing elements from $\mathbb{F}$. **All that matters is that the field operation axioms are satisfied**.
